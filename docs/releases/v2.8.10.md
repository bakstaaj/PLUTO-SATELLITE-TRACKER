# Pluto Satellite Tracker v2.8.10 - GMSK windowed timing analyzer

## Purpose

Add an offline GMSK Phase 2 diagnostic tool that scans captured PCM files in short windows to find the cleanest timing segment before attempting HDLC/AX.25 flag probing.

## Scope

- Adds `tools/analyze_gmsk_windows_v2_8_10.py`.
- Reads signed 16-bit little-endian PCM captured by the v2.8.8 harness.
- Evaluates 1, 2, and 5 second windows by default.
- Reports candidate symbol phase, timing score, phase-score spread, near-zero percentage, positive/negative balance, transition rate, RMS, peak, and clipping.
- Exports the best PCM window under `analysis/gmsk_phase2/windows/` for later slicer/HDLC experiments.

## Safety

This release does not touch the Pluto backend C source, does not rebuild the known-good deployed backend, and does not claim HDLC, AX.25, packet, or text decode.

## Example

```bash
python3 tools/analyze_gmsk_windows_v2_8_10.py \
  ../gmsk-phase2-capture-harness-v2.8.8/captures/gmsk_phase2/gmsk_capture_20260620T201209Z.pcm
```

Expected outputs:

- `analysis/gmsk_phase2/*_windowed_timing_v2_8_10.json`
- `analysis/gmsk_phase2/*_windowed_timing_v2_8_10.txt`
- `analysis/gmsk_phase2/windows/*_best_window_v2_8_10.pcm`
