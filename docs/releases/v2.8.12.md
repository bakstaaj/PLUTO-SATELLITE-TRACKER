# v2.8.12 - CW tone/envelope analyzer

Adds an offline/stream capture helper for CW receive diagnostics.

## Scope

- Adds `tools/capture_analyze_cw_pcm_v2_8_12.py`.
- Captures the existing Pluto live WAV stream or analyzes an existing PCM/WAV file.
- Reports broad zero-crossing tone, CW-band tone candidate, envelope threshold, key duty, estimated timing unit, tentative Morse symbols, and tentative text.
- Does not modify backend C.
- Does not claim reliable CW decode unless tone/envelope timing looks plausible.

## Usage

During a CW pass, start Receive in the UI, then run:

```bash
python3 tools/capture_analyze_cw_pcm_v2_8_12.py --host 192.168.68.104 --seconds 10
```

Artifacts are written under `captures/cw_v2_8_12/`.
