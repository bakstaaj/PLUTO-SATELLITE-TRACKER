# v2.8.28 - Single pass-row action modal fix

This UI-only patch fixes the pass-row backend-test dialog behavior introduced in v2.8.27.

## Changes

- Pass-row Listen/Receive backend-test clicks now suppress the older Receive placeholder modal so only one operator dialog opens.
- The older Receive placeholder modal receives explicit dark-theme styling so it is readable if opened elsewhere.
- Adds browser console helpers for field diagnosis:
  - `window.plutoHideLegacyReceivePlaceholderV2828()`
  - `window.plutoSuppressLegacyReceivePlaceholderV2828(ms)`

## Scope

- Web UI only.
- No backend C changes.
- No radio endpoint changes.
