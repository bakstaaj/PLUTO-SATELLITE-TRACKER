# v2.8.17 - Pass action visible mode labels

Fixes pass-list action button labels so they match the visible transmitter mode shown in the pass row.

The action button now uses the literal mode from `pass.modes[0]` / row metadata instead of broad satellite/family hints. Examples:

- `AFSK` -> `Receive: AFSK`, not `Receive: APRS`
- `BPSK` -> `Receive: BPSK`, not `Receive: Packet`
- `GMSK USP` -> `Receive: GMSK USP`, not `Receive: Digital`
- `FSK` -> `Receive: FSK`, not `Receive: Packet` or `Receive: Digital`
- `DVB-S2` -> `Receive: DVB-S2`, not inferred from the satellite name
- `USB` -> `Listen`
- `DUV` -> `Receive: DUV`

This is a web UI only patch. Backend C and decoder behavior are unchanged.
