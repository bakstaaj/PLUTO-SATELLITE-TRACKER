# v2.8.18 - Pass action DOM label repair

Adds a last-mile DOM repair for pass-list action button labels.

The repair derives the action label from the visible pass-row mode text instead of satellite name, APRS/Packet/Digital/CW family classifiers, or catalog hints.

Expected behavior:

- `CW` -> `Receive: CW`
- `DUV` -> `Receive: DUV`
- `GMSK USP` -> `Receive: GMSK USP`
- `AFSK` -> `Receive: AFSK`
- `BPSK` -> `Receive: BPSK`
- `DVB-S2` -> `Receive: DVB-S2`
- `FSK` -> `Receive: FSK`
- `QPSK` -> `Receive: QPSK`
- `USB` -> `Listen`

This is web UI only. Backend C and decoder behavior are unchanged.
