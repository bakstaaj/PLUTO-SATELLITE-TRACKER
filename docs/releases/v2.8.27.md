# v2.8.27 Pass-row backend test modal

Adds an explicit backend-test dialog for pass-row Listen/Receive actions.

## Scope

- Web UI only.
- No backend C changes.
- Opens a visible modal immediately when a pass-row action is clicked.
- Shows satellite, action, mode, downlink, AOS/LOS and live status.
- Routes Receive modes to the existing Receive dialog.
- Routes Listen modes to the existing backend-owned audio stream path with Stop/Close controls.
- Keeps pass-row action buttons enabled for backend testing outside AOS/LOS.
