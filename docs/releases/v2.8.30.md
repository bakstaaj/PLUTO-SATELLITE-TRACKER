# v2.8.30 - Single Backend Test modal for pass-row actions

This UI-only patch cleans up the v2.8.27-v2.8.29 pass-row backend-test flow.

## Changes

- Intercepts pass-row Listen/Receive clicks before older row handlers.
- Opens only the Backend Test modal.
- Hides/removes the obsolete `Open Receive dialog` button.
- For Receive/data/CW modes, the modal now shows the selected backend target and decoder status instead of attempting to open the retired legacy Receive Decode placeholder.
- For Listen/voice modes, the modal starts the existing backend audio path and keeps Stop/Close/Copy controls.
- Keeps pass-row buttons enabled for backend testing outside AOS/LOS.

## Scope

- Web UI only.
- No backend C changes.
- No decoder endpoint changes.
